Explore JS


Live Demo of Explore JS

Explore JS is a 3d first person shooter game built on vanilla javascript and three.js. I utilized a lot of loaded textures, and constructor classes to randomly generate the map, making the world seem a little more familiar. The game play utilizes Pointer Lock contrtols to give the user a better smoother experience over all.

The hardest part about this project was designing the physics. I had to research a lot of different collision formulas and tweak them to the Three.js vectors. Another problem I ran into was trajectory and following a users position while keeping consistant velocity, but by normalizing the vector, and utilizing three.js built in math, I was able to ensure normalized speed, diminishing rendering errors, and making the game play more forgiving.

Space Explorers


*currently being updated for Heroku-22*

Space Explorers is a space themed role playing game based in the near future. It uses a MongoDB, Express, React front-end and Node.js. We utilize the google maps Image Type Api to create the planets with custom info markers and icons. The state is persisted using local storage and cookies, so a user can easily save progress, load their last player and play where they left off.
Players navigate a chosen world and story- fighting enemies, completing tasks, and developing their characters abilities and arsenal.

React on the pfront end allowed us to grab everything we needed form MongoDB to really optimize the RPG system. State is only updated in the database after the user submits changes, making the interface faster.

The battle utilizes stats and random probability that takes into account each players statistics in order to complete the RPG experience. The user will evntually have to level up the robot in order to defeat stronger enemies.

Moon Camp


Live Demo of MoonCamp

MoonCamp is the only site in the universe to book and host spots on the moons of our eathlings' Solar System. Want to explore Titan, Europa, or maybe stay close to home on our moon, MoonCamp is the app for you.

MoonCamps top priority is to make an app that is beautiful and quirky. By utilizing modals and animations, MoonCamp has some personality and style that flows through the universe like a nebula. We used both React CSS transitions and good old css keyframes. By utilizing a switch, we can change the background or modal child of the Modal

One of the fun aspects of moon camp is using Google Maps imageType to map out different surfaces of planets using Nasa's open source satelite images. By utilizing google maps api, moon camp can normalize the longitudes and latitudes of the and imgaeType, making it easy to plot markers for different moon sites to rent out, all with live updates depending on the bounds of the map.

MTA Tracker


*currently being updated for Heroku-22*

MTA Tracker is a basic NYC subway tracker with live updates every 2 seconds. It utilizes googles GTFS in order to get trip updates for certain trains. It is set up so a user can select stations and view when the next trains will arrive, as well as the rest of the scheduled trains for that specific station. This project utilizes a file reader to skim over a .txt file provided by MTA, and pushes out a hash datastructure that hold the station names, ids, and directions. The ajax request is a syncronous and utilises gtfs-bindings and protos to decode the response. Once data is received one can selec a certain station ID and filter all soptTime updates for that specific station.
